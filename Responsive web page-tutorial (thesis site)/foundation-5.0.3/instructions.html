Instructions

At first press Download Foundation to go to the download page. There you can choose to download essential, complete, custom or SASS versions. For the custom download there is a lot of options to customize how the Foundation package will be constructed. A good way would be to download the complete package to see all the features that Foundation offer and the just delete the elements that are not needed for the specific site.

After downloading the Foundation zip just unzip it and you have a folder which contains subfolders for different types of files and index.html just like any website. Foundation just comes with preset files so you don't have to build the website from scratch and the basic responsiveness is added to those files. The best way to start experimenting is to build your own content over the index.html file. Be careful if you modify the Foundation CSS or JavaScript files and remember to take backups before adjustments so that the previous settings are easy to restore if something goes wrong. A good tip would be to put the Foundation folder to WampServer www-directory although this is not needed because Foundation does not contain PHP-files.
