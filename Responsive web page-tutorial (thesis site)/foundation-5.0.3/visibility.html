Text visibility

You can use visibility classes to either show or hide content depending which device or screen size is used. Try to resize the window to see different text shown as the screen size adapts. You can either show or hide for example set class to show-for-small-only or hide-for-small-only. This example below uses show-for-device type code. You can also set oriental detection, touch decetion and accessibility. The code can be found on Foundation documents. Visibility Classes.

This text is shown only on a small screen. This text is shown on medium screens and up. This text is shown only on a medium screen. This text is shown on large screens and up. This text is shown only on a large screen. This text is shown on xlarge screens and up. This text is shown only on an xlarge screen. This text is shown on xxlarge screens and up.

Image visibility

You can also use the same code for another content such as images. To hide the image below for small devices only I just put the image inside div and set div class instead of p class as in the text example above. Try to resize the browser window to minimum size to see the image disappear. For responsive web design the ability to manage visibility is really convenient because you can make loading times faster for smaller devices if you hide some of the content. Take a look at the Visibility Classes document and add some code to your Foundation index.html to begin experimenting with different visibility possibilities. There is no need to modify css. Just add classes in the HTML page and Foundation takes care of the actual work of hiding or showing content depending on which code is used.
