Relative font sizes

Relative font sizes can be used for font responsiveness when designing responsive sites so that the font-size may vary according to screen size or browser window size. You can see an example of this at the media query site or the wedding store example. You can read more information about relative font sizes here.

Responsive images

The same responsiveness can be applied to images so that the image size depending on which device is used of how big the browser window is. The little code that does the trick is this one below:

img {

max-width: 100%;

}

That works in most cases. That code is actually included in Foundation package so the image scaling works immediately. Just resize the home page of this tutorial site to see the scaling.

There are couple of things however that need to be considered then thinking about image scaling in responsive websites and those things can be found here.
