Beginning words

Written by Jari-Matti Seppälä on May 3, 2014.

This website demonstrates responsive web design. The site uses the latest version of Foundation framework which is a popular CSS framework. Try to resize the browser window to see Foundation in action. The image is a free to use type image from stockvault website. Link.

Foundation is a great way to begin studying responsive web design because it is easy to get started with and can be easily modified to suit your ideas.

Site instructions

Written by Jari-Matti Seppälä on May 3, 2014.

To begin using foundation click 'Get Started' on the top navigation. The link will get you to the foundation installation page. 'RWD-basics' tab will give you info about responsive web design and links to more tutorials. 'Responsive site' tab goes to a page that has a wedding store example converted to responsive site on Foundation. The wedding store is a site thas has been used as an example during web page development course.

The image is a free to use type image from stockvault website. Link.
