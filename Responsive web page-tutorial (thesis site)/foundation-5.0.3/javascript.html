JavaScript

JavaScript is most commonly used as part of web browsers to add interactions to the site. JavaScript is also used outside of web pages—for example, in PDF documents, site-specific browsers, and desktop widgets. When desigining responsive sites JavaScript can be used for example to detect the device and then alter the page by for example adding a smartphone-optimized image. Reference to JavaScript-files in the HTML5-document is done similarly to CSS3-files. For example Foundation comes with many JavaScript-files and one example would be adding < script src="js/foundation/foundation.js"> (without spaces) to include foundation.js-file in the document. Another thing to do that JavaScript works correctly is to add:

{ < script >
$(document).foundation();
< / script > }

(again without spaces, the spaces here are for not adding the code for second time) before the closing < body>-tag of the HTML-page.
