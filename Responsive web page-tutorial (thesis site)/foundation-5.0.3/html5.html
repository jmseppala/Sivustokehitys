HTML5

HTML5 is the latest version of HTML hyper text markup language. HTML5 was created to simplify the coding process and at the same time to make the coding more logical. In addition to being the latest version of the HTML language, HTML5 is an umbrella term for the next generation of web apps and how functionality will be expanded with better markup (HTML), better style (CSS), and better interactivity (JavaScript). There are bunch of new elements added to HTML5 and the most important of them are article, aside, header, footer, nav, section, audio, video, embed and canvas. HTML5-document is started by simply typing < !doctype html > (without spaces, the spaces here are added to not confuse this HTML-document with the doctype added second time) so the syntax is much shorter compared to previous HTML versions.
