Framework

Frameworks are often used when building responsive web designs. At the time of reading this text you probably already know that this tutorial site is build on Foundation 5 framework which is a really popular and developing framework. Foundation is not the only one as there are tons of different kinds of frameworks available. Some other popular ones include Twitter's Bootstrap and Skeleton. The great advantage of frameworks is the site templates that they come with so there is no need to build the site from scratch and because of that using frameworks add speed to the site development process.
