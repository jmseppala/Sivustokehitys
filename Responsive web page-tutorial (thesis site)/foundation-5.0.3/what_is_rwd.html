What is Responsive Web Design?

Responsive web design is a term used which is used for web designs that will adapt resolution of users' device., ie, creating a website which layout adapts according to resolution of users' device. The screen size of the device varies such as iPhone screen compared to a desktop computer screen and naturally the size of the browser window can be adjusted aswell. The common tehcnique before was to build different versions of the site for desktop and mobile devices or alternatively make native apps for different mobile platforms. The mobile version would be then put to a subdomain such as http://m.somemobilesite.com

Because of the different versions the development process itself takes a lot of time and also maintaining the site. When making responsive sites the idea is to make only one version of the site that adapts according to the screen size so effectiveness of the site is greater.
