CSS3

CSS3 is the latest version of the Cascading Style Sheets language. The biggest difference between CSS2 and CSS3 is probably the separation of modules. CSS3 is divided into several documents which are called modules. Every single module has new capabilities, without hurting the compatibility of the previous stable release. Although there are more that 50 modules the 4 most important ones are media queries, namespaces, selectors level 3 and color. You can find more information about media queries on this tutorial site. Just choose media queries under the RWD basics link. For more information about CSS3 in general just click the links below. CSS3 is used with HTML5 markup language. A reference to CSS3-file from the HTML5-document is made just like before with older HTML and CSS-versions. For example < link rel="stylesheet" href="css/foundation.css" /> . Just delete the space after the first arrow. I added the space here to not confuse this HTML-document with the second css reference.
