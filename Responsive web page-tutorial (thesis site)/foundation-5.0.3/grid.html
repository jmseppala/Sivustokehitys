Grid

Grids are used as a basis for layout. There are two types of grids which are fixed grid and fluid grid. Fixed grid is used when the site width is defined with pixels so that the property has fixed pixel-based value. Fluid grid uses percentage-based values so that the responsiviness can be done more fluidly compared to fixed values. So fluid grid is mainly used in responsive web design because its ability to adapt. It is also easy to set Foundation to use different set of grids for different needs. You probably noticed how the content of this grid page is a little bit more narrow compared to other pages? That is because I changed div class to use "large-6 columns" instead of "large-9 columns". It is that easy to begin to modify the Foundation grid layout. You can also choose which kind of grid the Foundation package contains when you first install Foundation.
